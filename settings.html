Intelligence Artificielle (Groq / Llama 3)

Génère tes critiques instantanément grâce à Llama 3. 100% gratuit, sans carte bancaire, ultra-rapide. C'est la solution recommandée pour éviter les problèmes de quotas Google.

⚡️ Obtenir une clé Groq gratuitement (1 minute)

  1. 1
    Va sur la console Groq
    → console.groq.com/keys
  2. 2
    Connecte-toi simplement
    Utilise ton compte Google, GitHub ou Email.
  3. 3
    Crée ta clé
    Clique sur le bouton "Create API Key" et copie la clé (elle commence par gsk_...).
🗝️

Clé API TMDB

The Movie Database (TMDB) est gratuit. Crée un compte et génère une clé API pour activer la recherche automatique.

Comment obtenir une clé gratuite ?

  1. 1
    Crée un compte TMDB
    → www.themoviedb.org/signup
  2. 2
    Va dans les paramètres de ton compte
    Profil → Paramètres → API
  3. 3
    Crée une nouvelle application
    Choisis "Développeur" → remplis le formulaire (usage personnel)
  4. 4
    Copie ta clé API (v3 auth)
    Elle ressemble à : a1b2c3d4e5f6…
🍅

Clé API OMDb (IMDb & Rotten Tomatoes)

Récupère les notes officielles IMDb et Rotten Tomatoes sur les pages des films.

⚡️ Obtenir une clé OMDb (Gratuit)

  1. 1
    Va sur OMDb API
    → omdbapi.com/apikey.aspx
  2. 2
    Souscris à l'offre gratuite (FREE)
    Entre ton email et valide. Tu recevras un lien d'activation par email avec ta clé.
  3. 3
    Copie ta clé API
    Elle ressemble à : ab1234c...
🚀

Clé API NASA (AstroBoard)

Optionnel. Sans clé, DEMO_KEY est utilisé (limité à ~30 req/h). Obtiens une clé gratuite pour un accès illimité.

  1. 1
    Va sur → api.nasa.gov/#signUp
  2. 2
    Remplis le formulaire (prénom, nom, email). Tu reçois ta clé immédiatement.
